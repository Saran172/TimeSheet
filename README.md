## ‚ú® Features

- Log daily work hours with start time, end time, and task description
- **Real-time online users list** ‚Äì Displays currently active/logged-in team members (powered by [ASP.NET Core SignalR])
- Prevent overlapping time entries (validates both past and current days)
- Dynamic customer & project selection with filtering
- Secure **Forgot Password** flow using security questions
- Role-based screens (different UI/views for Admin, Manager, Employee)
- Full **CRUD** operations for Customers, Projects, and Employees
- Export timesheet data to multiple formats (PDF, Excel, CSV) from most screens
- Daily reminder notifications at 6 PM (currently disabled)
